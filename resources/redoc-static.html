The API furnishes data concerning the current, historical, and forecasted weather and Air Quality conditions for various locations in India.

v1_aqi

List the current air quality index of a location

API used to get the list of Current Air Quality Index Information for India

query Parameters
pincode
string

If pincode query attribute provided, response will contain the AQI record for that Geo Location.

state
string

If state query attribute provided, response will contain the AQI record for that Geo Location.

district
string

If district query attribute provided, response will contain the AQI record for that Geo Location.

page_ref
string

Pagination Reference Index

limit
integer <int32>
Default: 100

It display the list from the startFrom value till th value provided in the limit attribute. If limit is not provided it will provide till default value which is 100.

order
integer <int32>
Default: 1

The response order will be based on the value passed. ascending(0) or descending(1)

sort_by
string

The response would be sorted based on one of the attribute passed. Attributes are : eventTime, state or district.
